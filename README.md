# Markov-Chain-Monte-Carlo-Algorithms
The two Qmd files contain R implementations of ULA and SGLD respectively for the examples in Chapter 3 of the thesis.
'MNIST SGLD 3 & 5' contains python implementations of SGLD and VRSGLD for binary classifications as decribed in Section 4.2 of the thesis.
'MNIST SGLD' contains python implementations of SGLD and VRSGLD for multiclass classifications as decribed in Section 4.3 of the thesis.
